# Lunchwich
Met Lunchwich (naam nog in progress), kom je nooit te weinig aan sandwich recepten.
Deze website vind met jouw gekozen ingredienten, sandwich recepten uit zijn recepten database.
Je kunt zelf ook recepten uploading maar je moet wel daarvoor een account aanmaken.

Het idee met deze webesite is dat je een interactieve menu krijgt waarbij je ingredienten selecteerd.
Na dat je al je ingredienten hebt geselecteerd, geeft de website recepten terug waarbij je kan filteren op hoe veel ingredienten je mist.

Dit heb ik toegevoegd omdat ik anders super veel recepten moest toevoegen dus ik had er een brede filter optie toegevoegd zodat je meer recepten ziet.
De recepten zelf had ik geen tijd voor om op te zoeken of te bedenken dus ik heb chat gpt gebruikt om recepten te genereren.

Je kunt zelf recepten toevoegen als je een account aan maakt die dan toegevoegd worden aan de database.
Elke recept toont aan wie de recept heeft gemaakt.
